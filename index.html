Macam-Macam Button Bootstrap dengan Magazine Layout

1. Button Variants

Gunakan kelas btn-primary, btn-success, dll untuk menandai konteks aksi tombol.
Contohnya:
Primary – Aksi utama.
Success – Tindakan berhasil.
Danger – Peringatan atau aksi destruktif.

2. Outline Buttons

Gunakan kelas btn-outline-primary, btn-outline-success, dll untuk tombol dengan border tanpa latar warna solid.
Cocok untuk desain minimalis dan modern.

3. Disabled Buttons

Tambahkan atribut disabled pada tombol <button> atau kelas disabled pada <a> untuk menonaktifkan tombol.
Tombol akan terlihat tapi tidak bisa diklik.

Link Disabled

4. Button Sizes

Gunakan btn-sm untuk tombol kecil, btn-lg untuk tombol besar, dan default untuk ukuran normal.

5. Block Buttons

Gunakan d-grid pada container tombol agar tombol memenuhi lebar container (block button).
Atau gunakan kelas w-100 pada tombol untuk lebar 100%.

6. Toggle Buttons

Gunakan btn-check untuk input checkbox atau radio yang dikaitkan dengan label tombol.
Juga bisa gunakan tombol dengan atribut data-bs-toggle="button" untuk toggle sederhana.
